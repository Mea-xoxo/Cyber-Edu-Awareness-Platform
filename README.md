# 🛡️ Cybersecurity Awareness Platform

A Django-based web application designed to educate users on essential cybersecurity concepts in a simple and interactive way.

---

## 🎯 Purpose

This platform helps non-technical users understand how to stay safe online by learning about common cyber threats and how to prevent them.

---

## 🔐 Features

- 📚 Topic-based learning (Phishing, Malware, Password Security, etc.)
- 🧠 Interactive quizzes for each topic
- 🎨 Dynamic backgrounds per topic
- 🔍 Search functionality for topics
- 🧭 Simple and user-friendly interface

---

## 🧠 Topics Covered

- Phishing Awareness   
- Malware 
- Dos Attacks  
- Social Engineering *(more coming soon)*  

---

## 🛠️ Technologies Used

- Python (Django)
- HTML5
- CSS3
- SQLite (default Django database)
